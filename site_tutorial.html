Como configurar um Ambiente PHP Local para programar em PHP





O que é PHP?

O PHP é uma linguagem de programação open source, ou seja, de código aberto, utilizada por programadores e desenvolvedores para construir sites dinâmicos, extensões de integração de aplicações e agilizar no desenvolvimento de um sistema, além de também ter outras funcionalidades como a criação de scripts para linha de comando.
Em resumo, o PHP desempenha um papel fundamental na criação de aplicações web dinâmicas e é uma ferramenta valiosa para desenvolvedores que desejam construir sites interativos e funcionais, com a disponibilidade de codigo aberto e maior compatibilidade com bancos de dados.

Como usar o XAMPP para executar o PHP:

  1. Inicialmente, entre no site oficial do XAMPP (https://www.apachefriends.org/index.html) e faça o download arquivo de instalação.



  2. No momento da instalação do arquivo, aparecerá uma mensagem de afirmação para realizar alteração no dispositivo (clique em "Aceito"). Após essa etapa, será iniciado o instalador do programa, nesse instante, todas as abas que aparecerem clique em "Next".



  3. Finalizando todo esse procedimento, aparecerá\ uma janela para executar o XAMPP (aceite).



  4. Ao abrir o software, deve-se estartar o "Apache"


  5. Agora, no local em que foi instalado o aplicativo, entre na pasta "XAMPP" e logo depois na "htdocs" (que é onde localiza-se o servidor local), nela exclua tudo o que está lá dentro e crie um pasta para ser aberta no VSC



  6. Abra essa pasta criada no Viual Studio Code e, crie um novo arquivo com a extenção ".php"



  7. No navegador, digite "localhost" no campo de pesquisa em conjunto da pasta e do arquivo PHP
